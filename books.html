Advanced Mathematics for Grade 11

Chapter 1: Algebraic Expressions

Algebra is a branch of mathematics that uses symbols and letters to represent numbers and quantities in formulas and equations. The letters are called variables because they can represent different numbers.

1.1 Variables and Constants

In algebra, we use letters to represent unknown values. These letters are called variables. For example, in the expression 3x + 5, the letter x is a variable. Numbers that have fixed values are called constants. In the same expression, 3 and 5 are constants.

Variables allow us to write general rules. For instance, the area of a rectangle can be written as A = l × w, where A represents the area, l represents the length, and w represents the width. This formula works for any rectangle, regardless of its specific measurements.

1.2 Algebraic Expressions

An algebraic expression is a combination of variables, constants, and operations (like addition, subtraction, multiplication, and division). Examples include:

  • 2x + 3
  • 5y² - 2y + 7
  • (a + b)(a - b)

Expressions can be simplified by combining like terms. Like terms are terms that have the same variable raised to the same power. For example, in the expression 3x + 2y + 5x - y, the like terms are 3x and 5x, and 2y and -y.
